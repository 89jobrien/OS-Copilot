A base class for defining actions with optional attributes like description, name, timeout, and action type.

This class serves as a template for defining various actions that can be executed within a system. It encapsulates common properties that any action would require, such as a name, description, execution timeout, and the type of action. The class is designed to be extended by more specific action implementations.
